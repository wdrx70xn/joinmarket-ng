# JoinMarket-NG

JoinMarket-NG (Next Generation) is a modern Bitcoin CoinJoin implementation for improving transaction privacy. It enables users to collaborate in CoinJoin transactions to break the link between inputs and outputs, enhancing financial privacy on the Bitcoin network.

## Full Live Documentation

For full documentation, see [JoinMarket-NG Documentation](https://joinmarket-ng.github.io/joinmarket-ng/).

## Installation

See the [Installation Guide](https://joinmarket-ng.github.io/joinmarket-ng/install/) for detailed instructions.

## Main Modules

| Module | Description |
|--------|-------------|
| [jmcore](https://joinmarket-ng.github.io/joinmarket-ng/README-jmcore/) | Core functionality (cryptography, networking, protocol) |
| [jmwallet](https://joinmarket-ng.github.io/joinmarket-ng/README-jmwallet/) | Wallet management and Bitcoin operations |
| [taker](https://joinmarket-ng.github.io/joinmarket-ng/README-taker/) | CoinJoin transaction initiation |
| [maker](https://joinmarket-ng.github.io/joinmarket-ng/README-maker/) | Liquidity provision service |
